Guía de Usuario

Aplicación de Gestión de Biblioteca

1. Login

Antes de entrar a la aplicación, se mostrará una ventana de login. Debes ingresar los siguientes datos de acceso:

  • Dirección IP: La dirección del servidor.
  • Puerto: El puerto de conexión al servidor.
  • Usuario: El nombre de usuario para acceder a la base de datos.
  • Contraseña: La contraseña del usuario de la base de datos.
  • Nombre de la base de datos: El nombre de la base de datos.

Una vez que ingreses correctamente estos datos, podrás acceder a la ventana principal.

2. Ventana Principal

La ventana principal te muestra una tabla con la información de la biblioteca. En esta ventana puedes ver y gestionar los siguientes elementos:

  • Alumnos: Puedes añadir, modificar o eliminar registros de alumnos.
  • Libros: Puedes añadir, editar o dar de baja libros.
  • Préstamos: Registra los préstamos de libros, permitiendo ver qué libros están prestados y a qué alumnos.
  • Historial de Prestamos: Consulta el historial de los préstamos realizados.

En cada una de estas secciones, tienes las siguientes opciones:

  • Crear: Para añadir nuevos registros.
  • Modificar: Para actualizar información de registros existentes.
  • Eliminar: Para borrar registros (o dar de baja en caso de los libros) que ya no sean necesarios.

Idioma

La aplicación es multiidioma, puedes elegir el español o el inglés.

Si necesitas asistencia técnica, contacta al soporte de la aplicación o consulta el manual detallado de usuario.
