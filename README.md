# Título del proyecto

ProyectoBiblioteca

## Descripción

Aplicación para gestionar datos de una biblioteca. Trata alumnos, libros, préstamos e historial de préstamos.

## Primeros pasos

### Cómo usar

* Clona el repositorio en algún IDE y ejecuta el programa.
